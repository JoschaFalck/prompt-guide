Prompting im Unterricht – Grundlagen

Lehrkräftefortbildung Erstellt mit Claude Code von Joscha Falck joschafalck.github.io/prompt-guide

1. Was ist ein Prompt?

Ein Prompt ist die Texteingabe, mit der du eine KI wie ChatGPT, Claude oder Gemini anweist, etwas zu tun. Je präziser der Prompt, desto nützlicher das Ergebnis.

Analogie: Der Auftrag an eine Vertretungslehrkraft
Vage: „Macht was mit dem Text auf Seite 47." → KI entscheidet selbst.
Konkret: Klasse, Aufgabe, Zeitrahmen und Format angegeben → direkt verwendbar.

Ein guter Prompt ist: klar (eindeutig formuliert) · vollständig (alle nötigen Infos) · spezifisch (Format, Länge, Ton) · realistisch (KI-umsetzbar)

2. Die fünf Bausteine

Baustein Leitfrage Beispiel Pflicht?
Rolle Wer soll die KI sein? „Du bist Biologielehrerin …" Empfehlung
Aufgabe Was soll sie tun? „Erstelle einen Lückentext …" Ja
Kontext Was muss sie wissen? „Klasse 7, kennt Brüche noch nicht …" Sehr empfohlen
Format Wie soll's aussehen? „Als Tabelle, max. 200 Wörter" Empfehlung
Einschränkung Was soll sie vermeiden? „Keine Fachbegriffe ohne Erklärung" Bei Bedarf
Vollständiges Beispiel:
Du bist Biologielehrerin an einem Gymnasium. [Rolle] Erstelle einen Lückentext zur Fotosynthese. [Aufgabe] Klasse 7, Schüler:innen kennen Chlorophyll, aber nicht Calvin-Zyklus. [Kontext] 10–12 Lücken, Lösungskasten am Ende, einfache Sprache. [Format] Keine Fachbegriffe über das Vorwissen hinaus. [Einschränkung]

3. Prompt-Arten im Überblick

ArtWann?Schlüsselformulierung
Instruktion Klare, abgegrenzte Aufgaben „Erstelle / Schreibe / Erkläre …"
Rollenpriming Perspektivenwechsel, Expertensicht „Du bist ein/e …, der/die …"
Few-Shot Bestimmten Stil reproduzieren „Hier sind zwei Beispiele … Erstelle nun …"
Chain-of-Thought Komplexe Aufgaben, Mathematik „Denke Schritt für Schritt nach …"
Iterativ Immer – zum Verfeinern „Mach es kürzer / spezifischer …"

4. Modernes & dialogisches Prompting

Gutes Prompting heute ist kein einmaliger perfekter Prompt – es ist ein lebendiger Dialog:

Die 4 Prinzipien modernen Promptings
  1. Starten statt perfektionieren: Grob einsteigen, dann verfeinern.
  2. Trial & Error: Das erste Ergebnis ist ein Entwurf, kein Endprodukt.
  3. Kooperativ: KI als Denkpartner nutzen – sie hilft dir, dein Ziel zu finden.
  4. Speichern: Gute Prompts in einer persönlichen Bibliothek ablegen.
Reverse Prompting: Zeige der KI ein Ergebnis, das dir gefällt, und bitte sie, den Prompt dafür zu schreiben. So entsteht eine wiederverwendbare Vorlage ohne aufwändiges Formulieren.
„Hier ist ein Arbeitsblatt, das mir gut gefällt: [einfügen]. Schreibe mir den Prompt, mit dem ich ähnliche Blätter für andere Themen erstellen kann."

5. Tipps & häufige Fehler

✓ Die 7 wichtigsten Tipps
  1. Immer Zielgruppe nennen (Klasse, Vorwissen)
  2. Format explizit vorgeben (Tabelle, Liste, …)
  3. Nie beim ersten Ergebnis aufhören
  4. Vorwissen der Schüler:innen angeben
  5. Klare Verben verwenden (Erstelle / Erkläre …)
  6. KI-Output immer fachlich prüfen
  7. Lange Prompts sind kein Problem
✗ Die 5 häufigsten Fehler
  1. Zu vage – kein Kontext, keine Zielgruppe
  2. Kein Format angegeben
  3. Beim ersten Ergebnis stehenbleiben
  4. KI-Output ungeprüft übernehmen
  5. Die Schüler:innen vergessen
Checkliste vor dem Abschicken: ☐ Aufgabe klar formuliert?   ☐ Zielgruppe angegeben?   ☐ Format spezifiziert?   ☐ Kontext vollständig?   ☐ Einschränkung nötig?
